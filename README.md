# 🎫 Support Ticket System API

API REST para gerenciamento de chamados (Help Desk)

## 🚀 Tecnologias
- Node.js
- TypeScript
- Prisma ORM
- MySQL
- JWT

## 🔐 Funcionalidades
- Registro e login de usuários
- Autenticação com JWT
- Criação de tickets
- Rotas protegidas

## ⚙️ Como rodar

```bash
npm install
npx prisma generate
npx prisma db push
npm run dev
